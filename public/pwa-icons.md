# PWA Icons

To add custom icons for the PWA:

1. Create `icon-192x192.png` (192x192 pixels) in the `public` folder
2. Create `icon-512x512.png` (512x512 pixels) in the `public` folder

For now, the app will work without icons, but you can add them later for a better experience.

You can use online tools like:
- https://favicon.io/
- https://realfavicongenerator.net/

Or create simple colored squares with text using image editors.
